Lab Computer Management

Thanks to a mentorship set up by University of California - Merced, my team and I had the opportunity to work with Western Digital to help manage their hundreds of computers. Utulizing OsQuery on several lab computers, we store parsed computer data using Python into a MySQL database. We created a web application dashboard using PHP which allows connection to our database. The dashboard will display each computers information including time since boot, CPU, memory, and disk usage. The same information is displayed for each process as well. We created a Super Taskbar Manager allowing the lab manager to efficiently delegate the computers to their users.
